## 1. Цветовая палитра

### 1.1. Первичные цвета (идентичность)

Эти четыре цвета формируют хроматический треугольник плюс акцент фестиваля.

| Образец | Переменная | HEX | Название | Назначение |
|---------|-----------|-----|----------|------------|
| 🟤 | `--c-brown` `--c-bg` | `#40241C` | Festival Brown | Основной фон сайта. Цвет земли, кирпича, индустриального бетона. Доминирует на 70% поверхностей. |
| 🔴 | `--c-red` `--c-red-bright` | `#E30513` | Energy Red | Акцентный цвет. Энергия, эмоция, опасность. Кнопки, активные состояния, ссылки, бордеры. |
| ⚪ | `--c-paper` | `#F5ECD9` | Dirty Paper | Белый бумажный (состаренный). Текст на тёмных фонах, секции editorial. Создаёт ощущение печатной продукции. |
| 💛 | `--c-yellow` | `#ffde00` 💛 | Signal Yellow | Сигнальный жёлтый. Выделение, внимание, акцент. Контрапункт к красному. Используется в тегах "Now Playing", рукописном теглайне и активных состояниях. |

**Правила использования первичных цветов:**
- Festival Brown — фон по умолчанию для всех страниц
- Energy Red — только для акцентных элементов (кнопки, бордеры, hover)
- Dirty Paper — только для текста и инвертированных секций
- Не использовать красный для фоновых блоков большого размера
- Не использовать бумажный цвет на светлых фонах (потеря контраста)

## 2. Типографика

### 2.1. Основной шрифт (идентичность)

| Свойство | Значение |
|----------|----------|
| **Шрифт** | Barlow Condensed |
| **Переменная** | `--f-display` |
| **Начертания** | Light (300), Regular (400), Medium (500), Bold (700) |
| **Кернинг** | `0.02em` (стандарт), `0.22em–0.32em` (UpperCase) |
| **Трансформация** | Всегда `uppercase`, кроме специальных случаев |
| **Интерлиньяж** | `0.95` (заголовки) |

**Где используется:**
- Все заголовки `h1`–`h6`
- Навигация (`header .nav a`)
- Кнопки и теги
- Метаданные, время, цены
- Статистика форума
- Футер

**Характер:** Узкий, промышленный, монументальный. Напоминает советские плакаты, уличные указатели, машинную маркировку.

### 2.2. Вторичный шрифт (текст)

| Свойство | Значение |
|----------|----------|
| **Шрифт** | Libre Caslon Display |
| **Переменная** | `--f-body` |
| **Начертания** | Regular (400) |
| **Размер** | `17px–19px` (параграфы) |
| **Интерлиньяж** | `1.7–1.85` |

**Где используется:**
- Основные абзацы
- Editorial-секции
- Текст карточек
- Формы
- Юридические документы

**Характер:** Display с засечками, тёплый, монументальный. Привносит классическую типографскую элегантность в длинное чтение. Утончённая текстура с засечками контрастирует с индустриальной Barlow Condensed, усиливая редакционную идентичность фестиваля.

### 2.3. Акцентный шрифт (рукописный)

| Свойство | Значение |
|----------|----------|
| **Шрифт** | Rock Salt |
| **Переменная** | `--f-accent` |
| **Начертания** | Regular (400) |
| **Трансформация** | `none` (нативный кернинг) |

**Где используется (максимум 2–3 раза на страницу):**
- Tagline в hero: "a love letter to the loud part of the 2000s"
- Заголовки на странице About (`accent-script`)
- Специальные акценты

**Характер:** Рукописный, DIY, панк-фанзин. Нарушает строгость системы.

### 2.4. Стек загрузки

```css
--f-display: "Barlow Condensed", "Arial Narrow", system-ui, sans-serif;
--f-body:    "Libre Caslon Display", "Times New Roman", serif;
--f-accent:  "Rock Salt", "Permanent Marker", "Comic Sans MS", cursive;
```

Шрифты загружаются через Google Fonts с `preconnect` для оптимизации.

## 3. Логотип

Логотип представляет собой SVG-изображение, расположенное в `assets/logo/logo.svg`.

- В хедере отображается как `<img>` рядом с текстовой плашкой "MetalHolic"
- В `assets/logo.svg` — дополнительная версия (возможно, устаревшая)
- Монохромный, без градиентов
- Навигация использует логотип как якорь на главную

## 4. Текстура и эффекты

### 4.1. Шум (Grain Texture)

```css
--grain: url("data:image/svg+xml;utf8,<svg...feTurbulence.../>");
```

Наложение шума поверх всего сайта:
- `opacity: 0.25`
- `mix-blend-mode: overlay`
- Фиксированное позиционирование (`z-index: 9999`)

Создаёт эффект старой бумаги, аналоговой фотографии, VHS-шума.

### 4.2. Затемнение Hero

```css
.hero-overlay {
  background:
    radial-gradient(ellipse at 30% 30%, rgba(184,31,28,0.18), transparent 55%),
    linear-gradient(180deg, rgba(14,9,7,0.05) 0%, rgba(14,9,7,0.85) 80%, var(--c-bg) 100%);
}
```

## 5. Основные компоненты дизайна

### 5.1. Hero

- Высота: `100vh − header`
- Изображение: `object-fit: cover`, opacity 0.55
- Фильтры на изображении: `contrast(1.15) saturate(0.85) brightness(0.85)`
- Текст накладывается поверх градиента (снизу вверх)

### 5.2. Marquee (бегущая строка)

- Бесконечная анимация (`38s linear infinite`)
- Имена групп разделены символом `★`
- Дублированный HTML для бесшовного цикла

### 5.3. Timetable (расписание)

- Сетка: `110px repeat(12, 1fr)` — часовая шкала с 17:00 до 05:00
- Слоты позиционируются через `left: XX%` и `width: XX%` (ручной рассчёт)
- Headline-слоты: жёлтый акцент
- Обычные слоты: красный акцент

### 5.4. News Band (карусель новостей)

- Горизонтальный скролл с `scroll-snap-type: x mandatory`
- Градиентные превью по категориям (atmos, chaos, legacy, press)
- Адаптивная ширина карточек

### 5.5. Forum (phpBB)

- Табличная вёрстка с фиксированными колонками
- 4 типа ячеек: иконка, тема, количество, последний пост
- Аватар с инициалом, ранг, дата регистрации

## 8. Тон коммуникации

- **Язык:** Английский (сайт)
- **Тон:** Кураторский, редакционный, интеллектуальный, без пафоса
- **Стиль:** Короткие абзацы, ёмкие формулировки
- **Аудитория:** 25–40 лет, ностальгия по нулевым, интерес к андеграунду
- **Правило:** "We treat the festival as an editorial archive — not a brand activation"

---

## 9. Запрещённые приёмы

- Не использовать градиенты кроме заданных
- Не использовать тени (box-shadow) — только бордеры
- Не добавлять закругления (border-radius) — только для Chip-фильтров (`999px`)
- Не использовать смайлики/эмодзи
- Не использовать изображения с людьми (кроме hero)
- Не использовать белый `#FFFFFF` — только `#F5ECD9` (paper)
- Не использовать чёрный `#000000` — только `#1D0F09` (deep brown)
- Не добавлять анимации кроме marquee и hover-эффектов
